A lightweight **Model Context Protocol (MCP)** demo server built with **Spring Boot** and **Spring AI**.  
This project exposes simple, server-side tools over **STDIO**, making it easy to integrate with MCP-compatible clients such as Claude Desktop.

---

## Features

- Minimal Spring Boot MCP server
- Tools exposed via STDIO using Spring AI MCP
- Clean, extensible tool definitions
- No HTTP layer required
- Ideal as a starting point for internal automation or custom enterprise tools
